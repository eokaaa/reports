Разработка эксплоита для vulnserver.exe

В этом экспоите использовалась уязвимость seh exception

В seh exception вызывается адрес памати, которые достает значения из стека и возвращается к коду в essfunc.dll.
После закидывает в код, в котором нужно сделать прыжок назад, чтобы в поместился код который хранится в reverseTCP.
После он выполняется и создает удаленное подключение.
